这是一个用于知乎网站的Chrome扩展，用于屏蔽网站首页的时间线上不感兴趣的内容。

因为知乎的设置，一个问题只能添加5个话题，而且屏蔽的话题也有上限，所以有时屏蔽了相关话题不能够保证问题不会出现在时间线上。

这个扩展会检测首页的每个答案，如果题目或答案中出现了你想要屏蔽的关键词，这个答案会默认隐藏，当然你也可以点击按钮重新查看答案。
